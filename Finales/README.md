# Modalidad coloquio

- Nota máxima: 6.
- Modalidad: oral.
- Preguntas frecuentes:
  - Dibujar el mapa de la computabilidad (los diferentes lenguajes, Diagrama de Venn).
  - Definir los diferentes tipos de lenguajes (RE, R, CO-RE, L, L_U, etc).
  - Preguntas de complejidad.
  - Máquinas de Turing, equivalencias, reducciones.
    - Definición de equivalencia.
    - Definición de reducción normal y polinómica. Para qué sirven.
  - "Algún lenguaje en RE puede tener una MT que se detenga para todas las entradas?" Sí, porque R es un subconjunto de RE.
  - "Todas las MT que reconocen un lenguaje de R se detienen para todas las entradas?" Falso, lo importante es que tenga al menos UNA MT que se detenga para todas las entradas, después podría tener equivalentes que loopeen.
  - Definir P y NP. En qué conjunto de lenguajes están incluidos?
